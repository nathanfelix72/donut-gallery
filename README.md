# Donut Gallery

A serverless web application showcasing donuts with AWS services.

## Architecture
- **Frontend:** HTML/CSS/JavaScript hosted on S3
- **CDN:** CloudFront distribution
- **API:** API Gateway + Lambda
- **Database:** DynamoDB
- **CI/CD:** GitHub Actions

## Deployment
Push to `main` branch to trigger automatic deployment to AWS.
